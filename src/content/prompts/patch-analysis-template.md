# パッチノート分析記事テンプレート

## 記事構成
1. **変更点サマリー**（400文字）: 今回のパッチの主要な変更点を箇条書きで。バフ/ナーフ/新要素に分類。
2. **各キャラクターへの影響**（1000文字）: バフ/ナーフされたキャラごとに変更内容と影響度を解説。ピック率・勝率への予測も。
3. **各武器への影響**（800文字）: 武器調整の内容と、使い勝手がどう変わるかを解説。
4. **新メタ予測**（600文字）: このパッチで環境がどう変わるか。強くなるキャラ/武器の予測とその根拠。
5. **おすすめ構成**（400文字）: 新メタでのおすすめチーム構成と武器セット。
6. **まとめ**（200文字）: プレイヤーが今すぐやるべきこと。
7. **CTA**: YouTube → note → LINE の順で配置。

## トーン
速報感を出す。「ついにきた」「大幅ナーフ」「環境激変」等のゲーマー的表現を使う。
ただしデータに基づかない誇張は避ける。

## SEO要件
- タイトルに「Apex パッチノート」「シーズンXX」を含める
- メタディスクリプションに主要変更点を1〜2個含める

## 文字数目安
3000〜4500文字（H2見出し5〜7個）
