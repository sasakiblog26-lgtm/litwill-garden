# ランク攻略型テンプレート

## 記事構成
1. **対象ランク帯の特徴**（400文字）: このランク帯のプレイヤーの傾向。よくある試合展開。敵のレベル感。
2. **勝つための立ち回り**（1200文字）: 降下場所の選び方→ローテーション→ファイト判断→最終リング。具体的なマップ例付き。
3. **ポイント計算の考え方**（400文字）: RP計算式と、このランク帯で盛るために必要な最低ライン。順位ごとのRP表。
4. **おすすめ構成**（600文字）: このランク帯で勝ちやすいチーム構成3パターン。バランス型・攻撃型・安定型。
5. **RP管理戦略**（500文字）: いつ攻めていつ引くかの判断基準。マイナスを最小化する考え方。
6. **まとめ**（200文字）: このランクを突破するための3ステップ。
7. **CTA**: YouTube → note → LINE の順で配置。

## H2見出し例（ゴールド→プラチナ攻略の場合）
- ゴールド帯の特徴と敵の傾向
- ゴールドを抜けるための立ち回り戦略
- RP計算を理解して効率よく盛ろう
- ゴールド帯で勝てるチーム構成3選
- RP管理の考え方：攻めと引きの判断
- まとめ：ゴールドからプラチナへの最短ルート
