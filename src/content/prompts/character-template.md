# キャラ攻略型テンプレート

## 記事構成
1. **キャラ概要**（300文字）: ロール、難易度、ピック率、勝率、どんなプレイヤーに向いているか。
2. **アビリティ解説**（800文字）: パッシブ・タクティカル・アルティメットそれぞれの効果と使いどころ。クールダウンや数値も記載。
3. **立ち回りの基本**（1000文字）: 序盤（降下・漁り）→中盤（移動・ポジション取り）→終盤（最終リング）での立ち回り。
4. **おすすめ武器構成**（500文字）: このキャラに合う武器セット3パターン。近距離特化・バランス・中遠距離型。
5. **味方との相性**（500文字）: 相性が良い味方キャラ3選と、なぜ相性が良いかの解説。
6. **よくあるミス**（400文字）: 初心者がこのキャラで犯しやすいミス3〜5つ。
7. **上級テク**（400文字）: 中級者以上が使える応用テクニック2〜3つ。
8. **まとめ**（200文字）: このキャラを使いこなすための3ステップ。
9. **CTA**: YouTube → note → LINE の順で配置。

## H2見出し例（レイス攻略の場合）
- レイスってどんなキャラ？基本情報まとめ
- アビリティ完全解説（虚空・ポータル・声）
- レイスの基本的な立ち回り
- レイスに合うおすすめ武器セット3選
- レイスと相性抜群な味方キャラ
- 初心者レイスがやりがちなNG行動
- 上級者向け：レイスの応用テクニック
- まとめ：レイスで勝率を上げる3ステップ
