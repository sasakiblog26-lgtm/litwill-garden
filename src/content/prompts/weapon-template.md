# 武器攻略型テンプレート

## 記事構成
1. **武器スペック**（400文字）: ダメージ、DPS、レート、弾薬タイプ、マガジンサイズ。表形式推奨。
2. **リコイルパターン**（500文字）: リコイルパターンの図解説明。どの方向にどれくらいブレるか。
3. **制御方法**（800文字）: リコイル制御のコツ。マウス/パッドそれぞれの操作方法。射撃訓練場での練習メニュー。
4. **おすすめアタッチメント**（400文字）: 優先度順にアタッチメントを解説。バレル→マガジン→ストック→スコープ。
5. **有利な交戦距離**（400文字）: この武器が最も活きる距離帯と、なぜその距離が有利なのか。TTKの比較も。
6. **プロの使用率**（300文字）: 競技シーンでの使用率と、プロがこの武器を選ぶ理由。
7. **まとめ**（200文字）: この武器を使いこなすための練習プラン。
8. **CTA**: YouTube → note → LINE の順で配置。

- R-301のリコイルパターンを理解しよう
- リコイル制御のコツと練習方法

- R-301が最強な交戦距離とは
